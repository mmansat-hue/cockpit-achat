Renseigne un véhicule pour comparer les prix
Le cockpit affiche le prix de vente sur ton stock actuel, sur ce que tu as vendu et chez tes concurrents, pour un véhicule similaire.
Brancher tes données Google Sheets. Colle l'URL CSV publiée de chaque feuille, puis « Charger ». Les données sont lues à chaque ouverture et mises en cache dans ce navigateur. Pour publier : dans Google Sheets → Fichier → Partager → Publier sur le web → choisir l'onglet et le format CSV.
Si une feuille ne se charge pas : un simple lien « edit » ou une feuille privée ne peuvent pas être lus par le navigateur. Deux solutions au choix, dans Google Sheets :
1. Publier : Fichier → Partager → Publier sur le web → choisis l'onglet + format CSV → Publier, puis colle l'URL obtenue.
2. Partager : bouton Partager → Accès général → Tous ceux qui ont le lien → Lecteur ; tu peux alors coller le lien « edit » directement (l'outil le convertit).
Prix d'achat : tes feuilles VN/VO contiennent des prix d'achat. Publie/partage plutôt un onglet « Vue Cockpit » sans ces colonnes, ou passe par n8n pour les garder privés.
